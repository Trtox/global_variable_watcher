Finds the address and size of a symbol in a binary(ELF). This function parses the symbol table of the given binary to locate the specified symbol.

Parameters
pathPath to the executable.
symbolName of the symbol to search for.
addressOutput parameter that will hold the symbol's address if found.
sizeOutput parameter that will hold the size (in bytes) of the symbol if found.
Returns
true if the symbol was found successfully, false otherwise.
